# 🍽️ Pooja Veg Catering Website

A modern and responsive catering service based website built using **HTML, CSS, and JavaScript**.
Designed to showcase catering services and allow users to easily send inquiries via WhatsApp.

---

## Features

* ✅ Fully responsive design (mobile, tablet, desktop)
* ✅ Modern UI with smooth animations
* ✅ Service cards with hover effects
* ✅ Contact form with WhatsApp integration
* ✅ Clean and user-friendly layout

---

## Sections Included

* Home
* About
* Services
* Contact

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Responsive Design)
* JavaScript (DOM Manipulation)

---

## 📬 Contact Form Integration

The contact form redirects user input directly to WhatsApp with a pre-filled message.
